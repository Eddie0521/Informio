## 开始之前

Informio 复用你本地已在运行的 Agent。请先安装并登录至少一个受支持的 CLI。

## Claude Code

1. 安装 [Claude Code](https://www.anthropic.com/claude-code) 并在终端完成登录。
2. 打开 Informio **设置 → Agent**。
3. 选择 Claude Code 作为当前运行时。

## Codex

1. 安装并登录 [Codex](https://openai.com/codex/)。
2. 在 Informio 设置中选择 Codex 作为运行时。

## OpenCode

1. 安装 [OpenCode](https://opencode.ai/) 并完成登录。
2. 在 Informio 设置中选择 OpenCode。

## 第一次使用 Agent

1. 打开 Markdown 文件。
2. 选中一段文字，或将光标放在希望 Agent 看到的上下文中。
3. 打开 Agent 面板并发送请求。

若运行时启动失败，Informio 会说明原因与下一步（例如 CLI 未安装或登录过期）。
