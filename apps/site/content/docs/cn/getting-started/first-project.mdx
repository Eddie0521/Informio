## 添加工作区

1. 启动 Informio。
2. 添加你正在使用的项目文件夹。Markdown 文件会留在原来的位置。
3. 从文件列表打开文档。

## 写作流程

- 中心编辑区是默认焦点。
- 只需写作时，可折叠左侧文件库与右侧 Agent 面板。
- 使用标签页在已打开文档间切换。

## 媒体与 PDF

Informio 可在同一工作区预览 PDF、图片、音视频、表格与 Word 文档。PDF 高亮可从 PDF 工具栏启用。

## 需要 Agent 时

请确认本地 Agent CLI 已安装并登录，然后在设置中选择。请参阅 [Agent 配置](/cn/docs/agent/setup)。
