Informio 是一款极简桌面 Markdown 编辑器，带上下文 Agent 辅助。中心编辑区是产品本身，Agent 紧贴写作，不喧宾夺主。

## 你可以做什么

- 添加项目文件夹，无需把文件搬进固定目录
- 在一个工作区审阅 Markdown、PDF、图片与媒体
- 用口令锁定敏感笔记
- 让本地 Agent 基于选中文本或当前文档工作
