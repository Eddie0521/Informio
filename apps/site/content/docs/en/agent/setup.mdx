## Before you start

Informio reuses Agents you already run locally. Install and sign in to at least one supported CLI first.

## Claude Code

1. Install [Claude Code](https://www.anthropic.com/claude-code) and complete login in Terminal.
2. Open Informio **Settings → Agent**.
3. Select Claude Code as the active runtime.

## Codex

1. Install and authenticate [Codex](https://openai.com/codex/).
2. In Informio settings, choose Codex as the runtime.

## OpenCode

1. Install [OpenCode](https://opencode.ai/) and sign in.
2. Select OpenCode in Informio settings.

## First Agent action

1. Open a Markdown file.
2. Select a passage or place the cursor in context you want the Agent to see.
3. Open the Agent panel and send a request.

If a runtime fails to start, Informio should show what happened and the next step (for example, missing CLI or expired login).
