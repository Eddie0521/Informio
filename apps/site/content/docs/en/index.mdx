Informio is a minimal desktop Markdown editor with contextual Agent assistance. The center writing surface is the product. Agents stay adjacent to writing, not louder than writing.

## What you can do

- Add project folders without moving files into a fixed directory
- Review Markdown, PDF, images, and media in one workspace
- Lock sensitive notes with a passphrase
- Run local Agents against selected text or the open document
