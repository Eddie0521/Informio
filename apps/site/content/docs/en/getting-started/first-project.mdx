## Add your workspace

1. Launch Informio.
2. Add the folder you are working in. Markdown files stay where they already are.
3. Open a document from the file list.

## Writing flow

- The center editor is the default focus.
- Collapse the left library and right Agent panel when you only need to write.
- Use tabs to switch between open documents.

## Media and PDFs

Informio can preview PDFs, images, video, audio, spreadsheets, and Word documents in the same workspace. PDF highlighting is available from the PDF toolbar.

## When you need Agent help

Make sure your local Agent CLI is installed and signed in, then pick it in Settings. See [Agent setup](/docs/agent/setup).
