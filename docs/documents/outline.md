# Outline

The outline panel provides quick navigation for Markdown documents by listing all headings.

## How it works

When you open a Markdown document, Informio automatically extracts headings (h1 through h6) and displays them in a hierarchical list. Indentation reflects heading levels — an h3 under an h2 is nested one level deeper.

## Jump to heading

Click any heading in the outline to scroll the editor to that position.

## Active heading tracking

As you move your cursor through the document, the outline highlights the heading closest to the current cursor position. This gives you a live sense of where you are in the document structure.

## Empty state

If the document contains no headings, the outline shows an empty state message.
