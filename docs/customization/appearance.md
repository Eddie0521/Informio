# Appearance

Controls for window layout, panels, and general UI chrome. Found across **Settings → Appearance**.

## Language

Switch the entire UI between Chinese and English. Found in **Settings → Appearance → Language**.

- **zh-CN** — Chinese (Simplified)
- **en-US** — English

The change applies immediately to all menus, labels, and system text. Your preference is saved and persists across sessions.

## Panel defaults

Set the default state for the left (library) and right (Agent) panels when Informio opens.

| Setting | Options |
|---------|---------|
| **Left panel** | Collapsed or Expanded |
| **Right panel** | Collapsed or Expanded |

Both panels can always be toggled at runtime. These settings control only the initial state.

## Panel width

Each panel's width is adjustable in pixels:

- **Left panel**: 161–380 px
- **Right panel**: 240–520 px

Drag the panel edge at runtime, or set a precise value in settings.

## Status bar auto-hide

When enabled, the status bar at the bottom of the window hides automatically and reappears on hover or interaction. This gives the editor a few extra pixels of vertical space.

## Show title in window title bar

Controls whether the current document title appears in the OS window title bar. Some users prefer a clean title bar, especially when using macOS traffic-light buttons.

## Chat font size

Sets the text size for the Agent conversation panel. Separate from the editor font size because the Agent panel has different density and readability needs.

- **Range**: 10–18 px

## Tips

- Collapse both panels for a distraction-free writing mode.
- If the Agent panel feels cramped on a laptop, try reducing the right panel width and increasing the chat font size slightly.
- The left panel width also affects how much of the file tree is visible without scrolling.
