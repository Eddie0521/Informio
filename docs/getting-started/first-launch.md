# First Launch

When you open Informio for the first time, you'll see a clean workspace ready for writing — no account, no database, no setup wizard.

## What you see

The window is divided into three areas:

- **Center editor** — a Markdown editor with a sample document already open. You can start typing immediately.
- **Left library panel** — shows your project files and folders. Collapsible via the panel toggle.
- **Right agent panel** — where Agent conversations happen. Also collapsible.

Both side panels can be collapsed to give the editor full width. Click the panel toggle buttons at the edges of the window, or use keyboard shortcuts to show and hide them.

## Adding a project folder

1. Click the **Add Folder** button in the library panel.
2. Select any folder from your filesystem.
3. Informio indexes the folder's files and displays them in the library.

Your files stay in their original location. Informio does not copy, move, or reorganize them — the folder you selected is the working directory.

You can add multiple folders from different locations. Each one appears as a separate root in the library tree.

## Panels

Both side panels follow the same behavior:

- **Collapsed**: only a thin rail is visible along the window edge.
- **Expanded**: the panel slides open and the editor width adjusts.
- **State persists**: Informio remembers your panel layout between sessions.

The editor is always the center of the workspace. Panels provide optional context — collapse them when you want distraction-free writing.

## What you don't need

- No database to set up. Informio reads and writes standard Markdown files directly.
- No account or sign-in. Everything runs locally.
- No mandatory configuration. Open the app and write.

To configure Agent integrations or other preferences later, open **Settings** from the menu or the command palette.
