# Quick Capture

Quick capture lets you create a note instantly without leaving your current context. It opens a minimal Informio window with both side panels collapsed and an empty editor, ready for typing.

## Triggering quick capture

Press **Control+Space** from anywhere on your system — the global hotkey works even when Informio is not in the foreground.

The shortcut is configurable. Open **Settings → Shortcuts** and change the **Quick Capture** binding to any key combination you prefer.

## What happens

1. A new Informio window opens.
2. Both the left library panel and right agent panel are pre-collapsed, giving the full window width to the editor.
3. An empty Markdown document is ready for input.
4. Start typing immediately — no file picker, no navigation.

The quick capture window is a regular Informio window. Once you are done, you can save, expand panels, or move the note into a project like any other document.

## Quick folder

New notes created through quick capture are saved to the **quick folder** — a configurable directory that acts as the default location for new documents. By default this is `~/Documents/Informio Quick Notes`.

You can change the quick folder in **Settings → Shortcuts → Quick Folder**. Use the folder picker to select any directory on your system.

## Workflow tip

Quick capture is designed for fleeting thoughts, meeting notes, and anything you want to write down without breaking your flow. Later, you can drag the note from the quick folder into a project folder to organize it.
